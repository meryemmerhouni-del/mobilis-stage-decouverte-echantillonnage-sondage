## Data Sampling Solution for Survey Launch

This project was developed as part of a **discovery internship** at **ATM Mobilis**, within the **Information Systems Department (DSI)**.

## Project Objective
The objective of this project is to design and implement a **data sampling solution** that enables the selection of a representative subset of customers for the purpose of launching surveys.

##  Host Company
**ATM Mobilis**  
Information Systems Department (DSI)

## Project Description
The application allows:
- Secure user authentication
- Uploading customer data files (Excel format)
- Filtering data based on multiple criteria (e.g., gender, region, offer, technology)
- Automatic calculation of sample size based on specified percentages
- Generation and regeneration of customer samples
- Export of sampled data in Excel format
